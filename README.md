# 电报群发助手

> 本人工作之余基于开源telethon开发的电报群发助手，需要下载请到release[下载地址](https://github.com/Cartwrights/telegram-sender/releases/download/tg/telegram-sender.zip)


目前实现的主要功能如下

1.批量注册

2.用户采集 群组采集

3.批量群发 批量拉人 批量私聊

4.自动吵群等
